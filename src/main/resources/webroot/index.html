📚 About

This server demonstrates:

  • @RestController — Component annotation for automatic discovery
  • @GetMapping — Maps GET requests to handler methods
  • @RequestParam — Binds query parameters to method arguments
  • IoC Container — Classpath scanning and dependency injection
  • Java Reflection — Runtime method invocation and annotation processing
